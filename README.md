# Survey Monkey API App

## Build with
- Ruby on Rails
- React Create App

## Get Started
### Requirements
- Ruby 2.7.4
- Rails 6
- Foreman gem

### Clone
```bash
git clone git@github.com:desoleary/survey-monkey-api-rails-react-app.git
cd survey-monkey-api-rails-react-app
```
### Install and migrate db

```bash
bundle install
yarn install --check-files
rails db:migrate
```

### Start Server
To start rails server
```bash
bin/rake dev:start

```
